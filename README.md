# INFNOVA Academy - Course Platform

A modern, responsive educational platform showcasing technology courses with a focus on Cloud Engineering with AWS.

## ✨ Features

- **Fully Responsive Design** - Optimized for mobile, tablet, and desktop
- **Interactive Course Cards** - Click on "Cloud Engineering with AWS" to view detailed course page
- **Live Search Functionality** - Search courses by title or instructor with debounce
- **Course Detail Page** - Comprehensive course information with sticky enrollment card
- **Mobile-Friendly Navigation** - Hamburger menu for mobile devices
- **Clean UI/UX** - Orange (#f97316) brand theme with proper spacing

## 🚀 Demo Navigation

1. **Homepage** (index.html) - Browse all 8 technology courses
2. **Click** on "Cloud Engineering with AWS" course card
3. **Course Detail Page** (course.html) - View comprehensive course information
4. **Click** "Back to Courses" to return to homepage
